# TDS-Q5: GitHub Actions Workflow with Status Badge

[![CI Test](https://github.com/23f2003821/TDS-Q5/actions/workflows/main.yml/badge.svg)](https://github.com/23f2003821/TDS-Q5/actions/workflows/main.yml)

## 📌 About This Project

This repository demonstrates a simple GitHub Actions workflow with a status badge.

The workflow runs automatically on:
- Every push
- Manual trigger using workflow_dispatch

## ⚙️ Workflow Details

- Runs on: Ubuntu latest
- Prints a success message
- Used to demonstrate CI/CD visibility

## 🚀 Purpose

This project was created as part of the IITM TDS assignment to show CI/CD transparency using GitHub Actions.
